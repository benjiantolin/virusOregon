Data Source & Attributions

This online interactive map enables users to track both Oregon State and County trends of Novel Coronavirus infection since Mar 18th, 2020. The supporting dataset is collected daily and then plotted onto this web application.

Data Sources

Oregon COVID-19 Data is collected from The Oregon Health Authority and then is plotted onto this web application.

You can view the data (in CSV format) at here.

In the data table, each entry indicates Cases, Deaths, and Negative Test Results in the format of "#-#-#".

County Geo Data collected from the U.S. Census Bureau Tiger/Line Shapefiles

Update Procedure
The Oregon state and county level COVID-19 data is currently being released daily from OHA. As data is released this application will continue to be updated.
